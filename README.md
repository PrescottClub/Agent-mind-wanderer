## 🚀 快速开始

### �� 环境要求
- Python 3.11+
- Windows 10/11 (优化支持)
- 4GB+ 内存
- DeepSeek API Key

### ⚡ 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/yourusername/Agent-mind-wanderer.git
cd Agent-mind-wanderer
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **配置API密钥**
```bash
# 复制环境变量模板
copy env_example.txt .env
# 编辑.env文件，添加你的DeepSeek API Key
```

4. **启动小念**
```bash
python main.py
```

5. **打开浏览器**
   - 访问：http://localhost:8501
   - 开始和小念的甜蜜对话 💕
